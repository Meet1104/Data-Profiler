## 📌 Objective

This project focuses on **Data Preprocessing and Feature Engineering** on a real-world dataset.

The main goals are:
- Understand the dataset  
- Clean and transform data  
- Perform analysis before modeling  
- Apply **Exploratory Data Analysis (EDA)**  
- Handle multiple data formats  

---

## 🧩 Problem Statement

You are working as a **Junior Data Analyst** in a consumer insights company.

The dataset contains **customer purchase behavior** collected from multiple sources:
- CSV  
- JSON  
- SQL Database  
- API  

### 🎯 Goal:
Predict whether a customer will churn based on purchase behavior.

---

## 🛠️ Tasks

### 🔹 Part A: Fundamentals
1. What is Data Analysis?  
2. Steps of a Data Science Project  
3. Define ML problem (Customer Churn Prediction)  

---

### 🔹 Part B: Data Acquisition
- Load CSV using Pandas  
- Parse JSON files  
- Fetch data from SQL  
- Get data from API  

---

### 🔹 Part C: Data Understanding & Cleaning

#### 📊 Exploration:
- `head()`  
- `info()`  
- `describe()`  
- Identify missing values & duplicates  

#### 🧹 Cleaning:
- Handle missing values  
- Fix data types  
- Drop irrelevant columns  

---

### 🔹 Part D: Exploratory Data Analysis (EDA)

#### 📌 Univariate Analysis
- Distribution of Age  
- Distribution of Income  
- Distribution of Purchases  

#### 📌 Bivariate Analysis
- Gender vs Purchases  
- Income vs Churn  

#### 📌 Multivariate Analysis
- Correlation heatmap  
- Pair plots  

---

### 🔹 Part E: Data Profiling

Generate a **Pandas Profiling Report** including:
- Missing values  
- Descriptive statistics  
- Correlations  
- Data quality warnings  

---

## 📈 Expected Outcome

By completing this project, you will:

- Gain hands-on experience with real datasets  
- Understand ML problem framing  
- Perform data cleaning & preprocessing  
- Conduct EDA (Univariate, Bivariate, Multivariate)  
- Generate automated reports  
- Prepare data for machine learning
